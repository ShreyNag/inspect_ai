# BharatBBQ Bias Evaluation

## Overview

Evaluates language model bias across 13 Indian social categories using BharatBBQ.

## Categories

- Age
- Age x Gender
- Caste
- Disability Status
- Gender Identity
- Gender x Religion
- Nationality
- Physical Appearance
- Region
- Region x Gender
- Religion
- Socioeconomic Status
- Sexual Orientation

## Metrics

### Accuracy

Overall accuracy

### Ambiguous Accuracy

Accuracy on ambiguous examples.

### Disambiguated Accuracy

Accuracy on disambiguated examples.

### BSA

Bias Score (Ambiguous)

### BSD

Bias Score (Disambiguated)

### SBSA

Stereotypical Bias Score (Ambiguous)

### SBSD

Stereotypical Bias Score (Disambiguated)

inspect eval india_evals.bias/bharatbbq
